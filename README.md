
### 访问接口服务

随后，可通过以下链接，访问默认接口服务。  
```
http://localhost/path/to/phalapi/public/
```
可以看到类似这样的输出：  
```
{
    "ret": 200,
    "data": {
        "title": "Hello PhalApi",
        "version": "2.0.1",
        "time": 1501079142
    },
    "msg": ""
}
```

> 温馨提示：推荐将访问根路径指向/path/to/phalapi/public。

更多请见：[PhalApi 2.x 开发文档](http://docs.phalapi.net/#/v2.0/)
